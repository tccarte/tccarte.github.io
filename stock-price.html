Solo Project

Stock Price Checker

A Flask web app that fetches live stock data from the Alpha Vantage API and renders daily, weekly, or monthly price charts directly in the browser as SVG.

▶ View Live App View on GitHub
Python Flask Matplotlib Alpha Vantage API SVG

What It Does

Users enter a stock ticker symbol (e.g., AAPL, TSLA) and choose a time interval — daily, weekly, or monthly. The app calls the Alpha Vantage API to retrieve historical price data, then generates a price chart and returns it inline as SVG so no image files are needed.

Invalid ticker symbols are caught before chart generation using the Alpha Vantage Company Overview endpoint, redirecting users back with a clear error rather than crashing on a bad API response.

Architecture

Built with Python and Flask. All data fetching and chart generation happens server-side — the client gets back a fully rendered HTML page with the SVG chart embedded directly. Matplotlib's non-interactive Agg backend is used to generate charts in memory without a display server.

User submits form ↓ Flask validates ticker via Company Overview API ↓ Fetch time-series data (daily / weekly / monthly) ↓ Matplotlib renders chart → SVG string in memory ↓ Return rendered HTML with embedded SVG

The key architecture decision was rendering charts as SVG strings in memory rather than saving image files to disk. This keeps the server stateless — no file cleanup needed, no storage cost, and SVGs scale perfectly at any screen resolution.

Key Features

  • Live stock data via Alpha Vantage REST API
  • Three time intervals: daily, weekly, monthly
  • In-memory SVG chart generation with Matplotlib
  • Ticker validation before chart generation to handle invalid symbols gracefully
  • Clean redirect flow — bad input goes back to the form, not a crash page
